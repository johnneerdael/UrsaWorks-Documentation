## What is UrsaWorks?

UrsaWorks helps you automate approvals and rejections of **URSA config requests**. You can:

- Build automation rules that match tenant type + config name
- Include or exclude specific requesters
- Execute rules in bulk (preview and run)
- Toggle feature flags for a tenant and auto-approve the request
- View run history and results
- Manage approved users (admin only)

## Quick Start

1. **Sign in** with your Google account (@netskope.com)
2. **Connect your URSA session** on the Dashboard
3. **Create rules** to automate approvals
4. **Run rules** manually or toggle features

## Features

- **Rules**: Automate approval decisions based on config, requester, and tenant
- **Manual Run**: Preview and execute your rules against open approvals
- **Feature Toggle**: Change feature flags and auto-approve requests
- **User Settings**: Control which configs are visible
- **Admin**: Manage user access and approval groups

## Security

UrsaWorks uses Google OAuth for authentication and requires URSA session cookies for operations. All data is handled securely with proper authorization checks.
